# Intelligent Vehicle Care AI - Predictive Maintenance Dashboard

## Project Overview

AI-powered predictive maintenance system for automotive OEMs. Monitor fleet health, predict failures, schedule appointments, and track agent activities in real-time.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server with auto-reloading and an instant preview
npm run dev
```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

- **Development server**: `npm run dev` - Starts the dev server on port 8080
- **Build**: `npm run build` - Creates a production build
- **Lint**: `npm run lint` - Runs ESLint

## Deployment

Build the project and deploy to your preferred hosting platform.
